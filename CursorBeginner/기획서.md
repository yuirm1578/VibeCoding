# 당신이 잠든 사이 (While You Were Sleeping)
## 미국주식 데일리 브리핑 서비스 기획서

---

## 1. 서비스 개요

"당신이 잠든 사이"는 미국주식 투자 한국인 직장인을 위한 데일리 브리핑 서비스입니다. 매일 밤 한국 시간 기준으로 미국 증시의 거래량과 상승률 데이터를 분석하여 가장 활발하게 움직인 종목을 자동으로 선정하고, 해당 종목의 핵심 정보와 관련 뉴스를 수집하여 AI가 생성한 이미지 브리핑으로 제공합니다. 사용자는 복잡한 분석이나 검색 없이도 매일 아침 이메일이나 슬랙을 통해 전날 밤 미국 증시에서 가장 주목받은 종목들을 한눈에 파악할 수 있습니다. 특히 바쁜 직장인 투자자들이 출근 전 짧은 시간에 잠든 사이 놓친 미국장 이슈를 빠르게 파악할 수 있도록 설계되었습니다.

---

## 1-1. 핵심 차별점

- **거래량 기반 자동 선정**: 복잡한 분석 없이 Yahoo Finance 거래량 데이터를 기반으로 가장 활발한 종목을 객관적으로 자동 선정
- **Chatless 자동화**: 사용자가 별도로 입력하거나 검색할 필요 없이 완전 자동으로 브리핑 생성 및 발송

---

## 2. 핵심 가치 제안 (Value Proposition)

### "잠든 사이 놓친 미국장 이슈를 아침에 한눈에"

한국 시간으로 잠든 사이에 벌어진 미국 증시의 모든 중요한 변화를 놓치지 않고, 아침에 일어나자마자 한눈에 파악할 수 있도록 제공합니다.

- **시간 절약**: 매일 수십 개의 종목을 직접 검색하고 분석하는 시간을 절약
- **거래량 기반 자동 선정**: Yahoo Finance 거래량 데이터를 기반으로 객관적이고 신뢰할 수 있는 화제 종목 자동 선정
- **Chatless 자동화**: 사용자가 별도로 설정하거나 검색할 필요 없이 완전 자동으로 브리핑 제공 (입력 없는 편의성)
- **시각적 이해**: AI 생성 이미지 브리핑으로 복잡한 데이터를 직관적으로 이해
- **한국인 맞춤**: 한국어로 제공되는 종목 정보와 뉴스로 언어 장벽 해소

---

## 3. 주요 기능 목록

1. **거래량 기반 자동 종목 선정 시스템**
   - Yahoo Finance API를 통한 실시간 거래량 및 상승률 데이터 수집
   - 거래량 증가율을 핵심 기준으로 화제 종목 자동 선정 (차별화 포인트)

2. **종목 정보 수집 및 정리**
   - 선정된 종목의 기본 정보(현재가, 전일 대비, 시가총액 등) 자동 수집
   - 관련 뉴스 기사 자동 수집 및 요약

3. **AI 이미지 브리핑 생성**
   - 종목 정보와 뉴스를 기반으로 AI가 시각적 브리핑 이미지 자동 생성
   - 차트, 핵심 지표, 요약 정보를 포함한 한눈에 보기 쉬운 형식

4. **Chatless 다채널 자동 발송**
   - 사용자 개입 없이 완전 자동으로 이메일 발송 (사용자 지정 시간)
   - Slack 웹훅을 통한 자동 알림 발송
   - 발송 시간 및 채널 사용자 커스터마이징 가능 (Chatless 자동화의 핵심)

5. **과거 브리핑 아카이브**
   - 발송된 브리핑의 과거 내역 조회 기능
   - 종목별 추적 및 성과 분석

6. **사용자 맞춤 설정**
   - 관심 종목 필터링 옵션
   - 브리핑 받을 시간대 설정
   - 선호하는 발송 채널 선택

7. **실시간 알림 (선택 기능)**
   - 급등/급락 종목 발생 시 즉시 알림
   - 사용자 설정 임계값 기반 알림

## 4. 사용자 시나리오 (User Journey)

### 시나리오: 직장인 김대리 (32세, 미국주식 투자 2년차)

**1단계: 서비스 발견 및 가입**
- 친구 추천 또는 SNS 광고를 통해 서비스 발견
- 간단한 이메일 주소와 Slack 워크스페이스 정보만으로 가입 완료
- 발송 시간 설정 (매일 오전 7시)

**2단계: 첫 브리핑 수신**
- 다음 날 오전 7시, 이메일과 Slack에서 첫 브리핑 수신
- AI 생성 이미지 브리핑으로 전날 밤 가장 활발했던 5개 종목을 한눈에 확인
- 각 종목의 상승률, 거래량 증가율, 관련 뉴스 요약을 빠르게 스캔

**3단계: 일상적인 사용**
- 매일 아침 출근 전 커피를 마시며 5분간 브리핑 확인
- 관심 있는 종목이 선정되면 상세 정보 클릭하여 추가 조사
- 브리핑을 통해 발견한 종목으로 투자 결정에 활용

**4단계: 맞춤 설정 활용**
- 특정 섹터(예: AI, 반도체)에 관심이 생겨 필터링 옵션 활성화
- 급등 알림 임계값 설정 (전일 대비 10% 이상 상승 시 즉시 알림)

**5단계: 장기 사용 및 추천**
- 3개월간 사용하며 투자 수익률 개선 체감
- 동료들에게 서비스 추천
- 과거 브리핑 아카이브를 통해 종목 추적 및 패턴 분석

## 5. 성공 지표 (KPI)

1. **사용자 참여도 (Engagement Rate)**
   - 일일 활성 사용자(DAU) 대비 브리핑 열람률
   - 목표: 70% 이상

2. **브리핑 유용성 지표**
   - 브리핑 내 종목 클릭률 (상세 정보 조회)
   - 목표: 브리핑당 평균 2개 이상 종목 클릭

3. **사용자 유지율 (Retention Rate)**
   - 가입 후 7일, 30일, 90일 유지율
   - 목표: 7일 유지율 60%, 30일 유지율 40%, 90일 유지율 25%

4. **자동 발송 성공률**
   - 이메일/Slack 발송 성공률
   - 목표: 99% 이상

5. **사용자 추천 지표 (NPS)**
   - Net Promoter Score (추천 의향 점수)
   - 목표: NPS 50 이상
