# 확장/자유 과제에 사용한 프롬프트 기록 (과제 5)

> 아래는 이번 작업에서 “확장 기능(과제 5)” 구현을 위해 사용한 프롬프트(요청) 기록입니다.

## 프롬프트 1 — 주가 차트 추가
```
현재 대시보드에 주가 차트를 추가해주세요.

[요구사항]
1. 최근 5일간 주가 추이 라인 차트
2. 거래량 바 차트 (하단)
3. 차트 라이브러리: recharts 또는 chart.js
4. 목업 데이터 사용
5. 상승/하락에 따른 색상 변경 (녹색/빨간색)

[위치]
TopStockCard 컴포넌트 내부 또는 별도 섹션
```

## 프롬프트 2 — TOP 3 비교 기능 추가
```
화제 종목 TOP 3를 비교할 수 있는 기능을 추가해주세요.

[요구사항]
1. 카드 3개를 나란히 배치 (1위, 2위, 3위)
2. 각 카드에 순위 뱃지 표시
3. 비교 항목: 주가, 변동률, 거래량
4. 호버 시 상세 정보 툴팁
5. 클릭 시 해당 종목 상세 페이지로 이동
```
