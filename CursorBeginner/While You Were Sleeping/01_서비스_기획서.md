# 서비스 기획서 — While You Were Sleeping

## 1) 서비스 한 줄 소개
미국 주식 시장에서 **하루 동안 가장 “화제”였던 종목을 자동 선정**하고, 사용자가 잠든 사이 핵심 정보를 **데일리 브리핑으로 요약**해주는 대시보드 서비스.

## 2) 문제 정의
- 해외 주식 투자자는 **밤 사이(미국장 시간대)** 발생한 정보를 아침에 빠르게 파악해야 함
- 수많은 종목/뉴스/차트 중 “오늘 꼭 봐야 할 종목”을 고르는 데 시간이 많이 듦
- 단순 가격 변동뿐 아니라 **거래량 급증, 변동성, 뉴스 이슈**를 함께 파악하기 어려움

## 3) 타겟 유저
- **미국주식 개인 투자자**: 출근 전 3~5분 내 핵심만 확인
- **관심 종목 추적자**: 거래량/변동성/뉴스 이벤트 기반으로 종목 후보를 탐색
- **초보 투자자**: 복잡한 지표 대신 요약 브리핑으로 이해

## 4) 핵심 가치(Why us)
- **화제 종목 자동 선정**: “오늘 볼 종목” 결정 피로도 감소
- **요약 브리핑**: 가격/거래량/이슈를 한 번에 확인
- **비교 기능**: TOP3를 나란히 비교해 “더 주목할 종목”을 빠르게 결정

## 5) 주요 기능(현재 구현/목업 포함)
### 5-1. 대시보드
- **오늘의 화제 종목(Top 1)** 카드
- **최근 5일 주가 추이 라인 차트 + 거래량 바 차트** (목업 데이터)
- **화제 종목 TOP 3 비교** (1/2/3위 카드, 상승/하락 색상, 호버 상세)
- **최근 브리핑 히스토리** 리스트

### 5-2. 종목 상세
- TOP3 카드 클릭 시 **`/stock/[symbol]`** 상세 페이지 이동(목업)

### 5-3. 브리핑
- 브리핑 상세 페이지 **`/briefing/[id]`** (목업)

## 6) 사용자 흐름(User Flow)
1. 사용자가 대시보드 접속
2. 화제 종목(Top1) 카드에서 현재가/변동률/거래량을 확인
3. 차트로 최근 5일 추이 확인 → 단기 방향성 파악
4. TOP3 비교 섹션에서 후보 종목을 비교
5. 관심 종목 클릭 → 종목 상세 페이지로 이동
6. 브리핑 히스토리에서 과거 요약 확인

## 7) 성공 지표(KPI) 제안
- DAU/WAU, 평균 세션 길이
- TOP3 카드 클릭률, 브리핑 상세 진입률
- “브리핑 생성” 버튼 클릭률(추후 API 연동 시)
- 재방문율(7일/30일)

## 8) MVP 범위
- MVP: “화제 종목 선정 → TOP 카드/차트/비교 → 브리핑 히스토리”까지
- 추후 확장: 실시간 데이터 연동, 뉴스/공시 요약, 알림/구독, 관심종목 저장
